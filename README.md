# ❤️ CardioSense  
### AI-Powered Heart Disease Prediction System  

CardioSense is a full-stack healthcare web application that predicts the risk of heart disease using machine learning models.

---

## 🚀 Features  

- 🔍 Heart Disease Prediction using ML models  
- 🏠 Dashboard  
- 📝 Prediction Form  
- 📊 Analytics Page  
- 🕒 History Page  
- 👤 Profile Page  
- 🔐 Authentication & Protected Routes  

---

## 🧩 Application Pages  

- **Dashboard** – Overview of predictions and quick insights  
- **Prediction Form** – Input health data and get predictions  
- **Analytics Page** – Visual charts for better understanding  
- **History Page** – Stores past predictions  
- **Profile Page** – User account management  

---

## 🧠 Machine Learning Models  

### Random Forest Classifier  
- Reliable for structured medical data  
- Provides strong baseline accuracy  

### LSTM (Long Short-Term Memory)  
- Captures deeper patterns in data  
- Improves prediction performance

---

## ⚙️ How It Works  

1. User fills in the prediction form  
2. Data is sent to the backend  
3. Backend processes the input  
4. ML models generate predictions  
5. Results are displayed on dashboard & analytics  
6. Data is stored for history tracking

## 🛠️ Installation & Setup  

```bash
git clone https://github.com/rosannechugh/CardioSense.git
cd CardioSense
pip install -r requirements.txt

cd backend
python app.py

cd ../frontend
npm install
npm run dev
```
